QGIS Safecast Plugin DocumentationÂ¶

This plugin allows to easily load data from Safecast radiation monitoring devices (LOG file) like bGeigie Nano into QGIS version 3 as a new vector point layer. The data are loaded using predefined color style.

The user can, if necessary, easily select and remove unwanted parts of the measurement set and save the corrected set as a new LOG file for uploading to Safecast web map.
